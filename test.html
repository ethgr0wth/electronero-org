Roadmap

2018 – Project Inception

Forked from Monero and Electroneum with enhanced privacy and security patches.

2019 – Mobile & Web Wallets

Released Android and Web light wallets for on‑the‑go payments.

2020 – RPC Libraries

Node.js, PHP, Go and Python modules added for easy RPC integration.

2021 – EI-2.0 Smart Chain

Smart Contracts, DeFi apps, Interoperability and Layer‑2 scaling research.

2022 – Multilingual UI

Added translations and streamlined build scripts for all platforms.

2023 – Packaging Improvements

Updated Docker images and RPC modules for easier deployment and integrations.

2025 – XRC-20 Token Platform

Robust token ecosystem accessible via CLI and RPC.

2025 – Governance Upgrades

Planned voting system and expanded token utilities.

Future Development

SFT Token Launch, creators can deploy Semi-Fungible Tokens

Future Development

Encrypted Private Messaging System integrated with CLI and RPC
